# Bug 描述引导

帮助你清晰地描述一个 bug。

## 请回答以下问题

### 1. 发生了什么？
> 你观察到的异常行为

### 2. 期望的行为是什么？
> 正常情况下应该怎样

### 3. 复现步骤
> 1. 第一步...
> 2. 第二步...
> 3. 出现问题...

### 4. 环境信息
> OS、Go 版本、相关依赖版本

### 5. 错误日志或截图
> 粘贴相关日志

### 6. 是否能稳定复现？
> 每次都能复现？偶发？特定条件下？

---

## 输出模板

```
## Bug: {简述}

**现象**: {异常行为}
**期望**: {正常行为}
**复现**:
1. ...
2. ...
**环境**: {环境信息}
**日志**: {错误日志}
**复现率**: {稳定/偶发/特定条件}
```
